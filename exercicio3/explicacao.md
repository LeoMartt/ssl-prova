A chave privada é um arquivo criptográfico usado para identificar de forma segura o proprietário de um certificado digital.

Ela deve ser mantida em sigilo, porque é a parte mais sensível do processo de criptografia assimétrica. Com ela, é possível assinar informações, descriptografar dados e gerar certificados.

Neste exercício, foi gerada uma chave RSA de 2048 bits chamada aluno.key
