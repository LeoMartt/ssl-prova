O comando scp (Secure Copy Protocol) é usado para transferir arquivos de forma segura por meio do protocolo SSH.

Neste exercício, o arquivo aluno.crt foi copiado usando localhost como destino, ou seja, para o próprio ambiente WSL. Essa abordagem permitiu demonstrar o funcionamento do scp mesmo sem o uso de uma segunda máquina física.

A principal vantagem do scp é que a transferência é feita de forma criptografada, protegendo o arquivo durante o envio.
